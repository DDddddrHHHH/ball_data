文件夹ball_data的数据基于定量相位成像技术由BioPhase system获取，内含70个聚苯乙烯微球对应的光程差（单位：m），像素100*100，对应面积为12×12μm2的实际视场 入射光波长632.8nm  样本直径4um/8um  折射率1.588，环境液为生理盐水（折射率1.33） 

上述数据由江苏大学生物光学及成像技术研究中心提供。欢迎访问江苏大学生物光学及成像技术研究中心（https://phy.ujs.edu.cn/info/1533/24592.htm）
